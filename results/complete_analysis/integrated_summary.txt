================================================================================
DENSE vs STATIC vs DPF: 통합 성능-프라이버시 분석
================================================================================

📊 실험 개요:
----------------------------------------
총 실험 수: 12
총 훈련 시간: 5.58 시간
평균 정확도: 90.34%

🎯 핵심 발견사항:
----------------------------------------
1. 유틸리티 (정확도):
   - Dense > DPF > Static (동일 스파시티에서)
   - 높은 스파시티에서 더 큰 성능 차이

2. 프라이버시 (MIA 저항성):
   - 스파시티 증가 → MIA 취약성 감소
   - DPF와 Static 간 프라이버시 차이 분석 필요

3. 효율성:
   - DPF: 더 나은 정확도-스파시티 트레이드오프
   - Static: 더 빠른 훈련 (Dead weight 업데이트 없음)

📈 권장사항:
----------------------------------------
• 최고 성능 필요: Dense 모델 사용
• 프라이버시 중요: 높은 스파시티 DPF
• 효율성 중요: 적당한 스파시티 Static
• 균형 필요: 70-80% 스파시티 DPF
